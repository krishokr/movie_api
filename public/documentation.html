Method Request URL Response
GET None /movies A JSON object holding data about all movies.
GET None /movies/:Title A JSON object holding data about the description, genre, director, image URL, actors, and whether it’s featured or not a single movie.
GET None /genres/:Genre A JSON object holding data about a genre (description) by name (e.g., “Thriller”).
GET None /directors/:Director A JSON object holding data about a director (bio, birth year) by name.
GET None /users A JSON object containing data about all users.
GET None /users/:Username A JSON object holding data about a user (Username, Email, Password, Birthday) by Username.
POST A JSON object holding data about a new user to add, structured like this: { "Username" : "Frodo Baggins", "Email" : "f.baggins@theShire.net", "Password" : "leadMe2Mordor!", "FavoriteMovies" : "There and Back Again" } /register A JSON object about the user that was just added, including: { "_id" : 12345, "Name" : "Frodo Baggins", "Email" : "f.baggins@theShire.net", "Password" : "leadMe2Mordor!", "FavoriteMovies" : "There and Back Again" }
PUT A JSON object with data the user wishes to update, including: { "Username": "f.bag", "Email": "fbag@gmail.com", "Password": "123", "Birthday": "1992-01-03" } /users/:Username A JSON object with the updated user's information.
PUT None /users/:Username/movies/:MovieID A JSON object with the user's updated movies.
PUT None /users/:Username/movies/:MovieID A JSON object with the user's updated movies.
DELETE None /users/:Username A text message letting the user know they have been unregistered.
