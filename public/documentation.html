Method Request URL Response
GET None /movies A JSON object holding data about the top movies.
GET None /movies/:title A JSON object holding data about the description, genre, director, image URL, and whether it’s featured or not a single movie.
GET None /genres/:genre A JSON object holding data about a genre (description) by name/title (e.g., “Thriller”).
GET None /directors/:director A JSON object holding data about a director (bio, birth year, death year) by name.
POST A JSON object holding data about a new user to add, structured like this: { "name" : "Frodo Baggins", "email" : "f.baggins@theShire.net", "favoriteMovie" : "There and Back Again", "password" : "leadMe2Mordor!" } /register A JSON object about the user that was just added, including: { "id" : 12345, "name" : "Frodo Baggins", "email" : "f.baggins@theShire.net", "favoriteMovie" : "There and Back Again", "password" : "leadMe2Mordor!" }
PUT None /accounts/:username/:name?/:email?/:password? A text message letting the user know their file has been updated.
PUT None /favoriteMovies/:username/:title A text message letting the user know their new favorite movie has been added.
DELETE None /favoriteMovies/:username/:title A text message letting the user know their movie has been deleted.
DELETE None /users/:username A text message letting the user know they have been unregistered.
